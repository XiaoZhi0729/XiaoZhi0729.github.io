Why Wallflip

同一张壁纸，不该陪你走过白天黑夜

手机会在浅色与深色模式间来回切换，但桌面壁纸却一成不变。亮色界面下偏暗的壁纸过于沉重，深色界面下偏亮的壁纸又格外刺眼。Wallflip 让这件小事，自己解决。

01

浅色与深色不适配

明亮壁纸在 Dark Mode 下过曝，深色壁纸在 Light Mode 下沉闷，无论选哪一张总有一半时间不对味。

02

手动换壁纸太麻烦

每次想要换个心情都要进相册、再进设置，几步操作下来兴致全无。久而久之，壁纸再好也会审美疲劳。

03

一致感被打断

系统主题、图标、UI 都在响应日夜，唯独壁纸停留在原地，整块桌面观感被瞬间割裂。

Features

把日常的壁纸切换，做到无声胜有声

四件你不用操心的小事，让桌面始终处在最舒服的那一面。

定时切换

按你设定的节奏自动更换壁纸，让桌面在不知不觉间常看常新。

Scheduled

日夜模式

系统切到 Dark Mode 时换上沉静的深色壁纸，回到 Light Mode 时立刻明亮起来。

Day & Night

Lightweight

适配体验

兼顾不同屏幕比例与系统版本，切换过程顺滑，不破坏锁屏与桌面之间的原生质感。

Screenshots

看看 Wallflip 是怎么把这件小事做轻的

五个核心界面，从首页到细节调节，每一步都尽量保持简单与克制，让设置壁纸像翻一页书那样自然。

01 / 05

首页一目了然

所有壁纸卡片整齐铺陈在主页，当前生效的方案、下一次切换的时间、日夜状态都在一屏内看完。无需多余跳转，桌面想要什么、Wallflip 已经在做什么，一眼就能心里有数。

Home
02 / 05

挑选一张，按下即换

点击卡片即可进入选择页，把想要的那张壁纸快速挑出。预览、切换、回到首页一气呵成，避免被层层菜单打断手感。

Pick
03 / 05

规则配置，节奏由你定

编辑规则页用来安排壁纸什么时候出场：定时切换、跟随系统日夜、按时段切换……每一条规则都是清晰的卡片，加减改都是一个动作的事。

Rules
04 / 05

单张壁纸的精细打磨

编辑壁纸效果面板专注于一张图：缩放、位置、显示偏好都能单独调，让同一张图在锁屏与桌面上都站得住。

Effect
05 / 05

能预判返回的手势动画

移动卡片与返回手势之间留有过渡：滑动时卡片会先做出反应，松手的瞬间已经知道你想去哪里。配合 Android 13+ 的预测式返回，整套动效连贯自然，仿佛壁纸本来就懂你下一步。
